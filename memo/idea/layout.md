# テキスト配置

Bashで文字列の配置をする。段組。

基本的にはコマンドを用いる。[memo/memo.md](memo/memo.md)参照。

## 枠(block)

```sh
+--------------+
|              |
| 任意テキスト |
|              |
+--------------+
```

```sh
┌──────┐
│任意テキスト│
└──────┘
```

### 引数

* 表示座標
* 幅,高さ
    * 折り返し
* 余白
* 内部位置（右寄せ、左寄せ、中央寄せ）


### 段組

+------------++----------+
|ひとつ目の  ||ふたつ目の|
|テキストです||テキスト  |
+------------+|です      |
              +----------+
+--------------------+
|３つめのテキストです|
+--------------------+

### 固定長

+------------+
| 項目名  値 |
| 項目名  値 |
| 項目名  値 |
| 項目名  値 |
+------------+

超過した部分の扱いを選択する。

* ぶった切る
* `…`, `~`
* 可変長にして枠のサイズを拡張する
